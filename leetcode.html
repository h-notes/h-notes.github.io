Count Square Submatrices with All Ones   NOTE

[2020-04-07 二 16:59]

leetcode

Given a m * n matrix of ones and zeros, return how many square submatrices have all ones.

Input: matrix = [ [0,1,1,1], [1,1,1,1], [0,1,1,1] ]

解法：找出递归关系，dp[i][j] 代表以 matrix[i][j] 做为右下角的最大 square 维数，同时也是以 matrix[i][j] 作为右下角的 square 个数。

我们有: 如果 matrix[i][j] = 1, dp[i][j] = min(dp[i-1][j-1], dp[i][j-1], dp[i-1][j]) + 1; 如果 matrix[i][j] = 0, dp[i][j] = 0。

最终，总的 square 数就是 所有的dp[i][j] 之和。
