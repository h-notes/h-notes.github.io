Count Square Submatrices with All Ones   NOTE

[2020-04-07 二 16:59]

leetcode

Given a m * n matrix of ones and zeros, return how many square submatrices have all ones.

Input: matrix = [ [0,1,1,1], [1,1,1,1], [0,1,1,1] ]

解法：找出递归关系，dp[i][j] 代表以 matrix[i][j] 做为右下角的最大 square 维数，同时也是以 matrix[i][j] 作为右下角的 square 个数。

我们有: 如果 matrix[i][j] = 1, dp[i][j] = min(dp[i-1][j-1], dp[i][j-1], dp[i-1][j]) + 1; 如果 matrix[i][j] = 0, dp[i][j] = 0。

最终，总的 square 数就是 所有的dp[i][j] 之和。

Minimum Cost Tree From Leaf Values   NOTE

[2020-04-08 三 10:50]

Given an array arr of positive integers, consider all binary trees such that:

  • Each node has either 0 or 2 children;
  • The values of arr correspond to the values of each leaf in an in-order traversal of the tree. (Recall that a node is a leaf if and only if it has 0 children.)
  • The value of each non-leaf node is equal to the product of the largest leaf value in its left and right subtree respectively.

Among all possible binary trees considered, return the smallest possible sum of the values of each non-leaf node. It is guaranteed this sum fits into a 32-bit integer.

目标

已知中序叶子节点值数组 arr，在上述构造完整树的步骤限制下，求 minimum cost tree value。这里简写成 mctv。 其中 ctv 是指树中所有内部节点之和。

思路

首先手动构造一个树，找出递归关系。

切分数组，计算切分之后左右两边数组的 mctv，那么切分之前的树组的 origin-mctv = left-mctv + right-mctv + max(left-arr) * max(right-arr)。

考虑所有的切分情况，返回最小的 ctv。

边界情况：当 array 长度为 1 时，ctv 等于 0。
